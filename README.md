# Live Cricket Score

Get Live Cricket Score on your Terminal and CMD  

## Usage

- Download or clone the repo

```sh
https://github.com/mskian/go-cricket-score.git
go-cricket-score
```

- Execute the Script

```sh
go run main.go
```

## Build Package

- Run Make file to build a package for your Systems

```sh
make build
```
